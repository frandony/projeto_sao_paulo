# São Paulo — Estética Urbana

Site de página única sobre a estética urbana de São Paulo, desenvolvido como projeto da disciplina (Etapas 1 e 2).

🔗 **Acesse o site:** https://frandony.github.io/projeto_sao_paulo/

## Sobre o projeto

O site explora o caos urbano e a estética crua de São Paulo através de vídeo, galeria de imagens e uma comparação com outras cidades (Curitiba, Vancouver e Tóquio).

## Tecnologias utilizadas

- HTML5
- CSS3 (estilo próprio + [Bootstrap 5.3.3](https://getbootstrap.com/))
- JavaScript (vanilla, sem bibliotecas)
- [GitHub Pages](https://pages.github.com/) para publicação

## Funcionalidades

**Etapa 1 — Estrutura e estilo**
- Layout responsivo com Bootstrap
- Banner principal com vídeo de fundo
- Galeria de imagens com efeito hover
- Tabela comparativa entre cidades
- Formulário de contato

**Etapa 2 — Interatividade com JavaScript**
- 🌙 **Modo claro/escuro**: alterna o tema do site e salva a preferência no navegador (`localStorage`)
- ✅ **Validação do formulário**: valida nome, email e telefone em tempo real, com mensagem de sucesso ao enviar
- 🔢 **Contador animado**: a seção "São Paulo em números" anima os valores ao rolar a página até ela
- ↕️ **Tabela ordenável**: clique nos cabeçalhos da tabela comparativa para ordenar as cidades

## Estrutura de arquivos

```
projeto_sao_paulo/
├── index.html       # estrutura da página
├── style.css        # estilos
├── script.js        # funcionalidades interativas
└── assets/          # imagens e vídeo
```

## Como executar localmente

Basta abrir o arquivo `index.html` em um navegador.
